     Faw Neek Restaurant Website
 - A Faw Neek Restaurant featuring authentic Kenyan cuisine. 
-Users can browse categorized menus, search for dishes in real-time, view prices with images and make a reservation
- The site includes an interactive star rating system on the homepage and menu page with search bar and booking reservation

 
 #Technologies used
1.HTML
2.CSS
3.JavaScript 
4.Font Awesome 6.0.0
5.External image hosting (Unsplash, Freepik ,pinterest)

   #Features
1.Animated homepage with star ratings
2.Real-time menu search functionality
3.Categorized menu display (Voyage, Main Dish, Local Dish, Starters)
4.Responsive reservation form with validation
